# Contributing to SyllabSync 📚

Thank you for helping improve SyllabSync! This project aims to make student life easier through open-source automation.

### 🛠️ Development Environment
SyllabSync is built using:
- **n8n** (Workflow Automation)
- **Groq AI** (LLaMA 3.3 models)
- **Google OAuth2** (Gmail & Calendar)

### 💡 How to Contribute
- **Workflow Optimizations:** If you can make the n8n nodes more efficient or reduce the number of executions, please submit a PR.
- **AI Prompt Engineering:** Improving the `Groq — Write Email` or `Groq — Generate Cal Events` prompts for better accuracy.
- **Documentation:** Help us improve the setup guide or add troubleshooting tips for different timezones.

### 🚀 Submitting Changes
1. **Export your workflow:** In n8n, go to **Workflows > Export to File**.
2. **Clean your data:** Ensure your API keys and personal emails are removed before committing the JSON.
3. **Open a PR:** Describe the changes you made and include a screenshot of the updated workflow canvas.
